חיבור ל-Google Drive

הנתונים שלך יישמרו בקובץ vendingbk.json ב-Drive שלך. גישה מהמחשב ומהטלפון, ללא שרת.

צריך ליצור פרויקט Google ב-console.cloud.google.com — לוקח כ-5 דקות.

שלב 1 — Client ID

1. כנס ל-Google Cloud Console
2. צור פרויקט חדש (כל שם)
3. APIs & Services → Enable APIs → חפש "Google Drive API" → Enable
4. Credentials → Create Credentials → OAuth 2.0 Client ID
5. Application type: Web application
6. Authorized JS origins: הוסף null (לקובץ מקומי)
7. העתק את ה-Client ID לכאן:

שלב 2 — התחבר

לחץ על הכפתור לאישור גישה ל-Drive שלך. האפליקציה תיצור קובץ vendingbk.json בתיקיית הבסיס.
